# Git에서 빈 디렉토리 추가하기

## 날짜
2026-03-09

## 배운 것
Git은 빈 디렉토리를 추적하지 않습니다.

## 상세 내용
git은 작업 영역, 인덱스 영역, 헤더 영역으로 나누어져 있습니다
이때 인덱스 영역은 파일 단위로 동작합니다 
즉, 파일이 없는 빈 디렉토리는 추적하지 않습니다

그럼에도 빈 디렉토리를 추가하고 싶다면
- '.gitkeep' 이라는 임의의 파일을 디렉토리 하위에 생성하고 추후 삭제합니다 (관례적인 방법)
- 'gitignore'이라는 파일을 디렉토리 하위에 생성하여 존재는 하되 내용물은 무시하도록 합니다
